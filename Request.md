## Login
1. 正常登录，登出
2. 登录后
    2.1 30秒没动作自动退出
    2.2 30秒期间有动作状态不断清0
    2.3 5分钟后提示用户是否需要继续停留在这个页面
        2.3.1 如果用户没有管，那么可能在计算的30秒后自动退出
        2.3.1 如果用户点击 继续 按钮，则续约5分钟
    2.4 10分钟后，提示用户由于安全强制退出
3. 接口超时
    3.1 当处于弱网时，由于接口超时，会触发30秒无动作，此时哪怕用户在30秒成功请求，也会由于 无状态 而自动退出
    3.2 当触发 3.1 的时候，会有一个 Loading... 此时可能产生页面变成了 Login 但是 Loading 仍旧存在，需要解决
4. 刷新
    4.1 刷新不影响登出，除非刷新遇到 3.1
    4.2 刷新也可能遇到情况 3
